# MatrixScreenLoader-Funnier
Just for fun

Command keys
  '/w=27' - set number of columns on horizontal;
  '/h=13' - set number of rows on vertical;
  '/t=50' - set timeout between refreshing screen in milliseconds; [50ms on default];
  '/c=true' - set if before each digit display clear the screen; [true on default];
  '/d=10' - set diapason of digits; 2 - only 0 and 1; NB! Do not set more then 10; [2 on default];
  '/r=true' - set if update width and height on resize;
  
Try to small windows, to get better effect;
